# 簡單實用工具
一個小型實用附加, 它增加了有用的工具與方塊.

此為**非官方**版本, 請勿在該作者的問題追蹤內回報!
[原作者連結](https://github.com/Mooy1/SimpleUtils)

## 特徵

- 增強工作台 - 製作原版與黏液科技配方在單個選單


- 礦槌錘 - 一次開採 3x3 或 5x5 範圍.


- 扳手 - 快速破壞物流節點, 電容, 和電力機器
