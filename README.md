# 簡單實用工具
一個小型實用附加, 它增加了有用的工具與方塊.

> 此為**非官方**版本, 請勿在該作者問題追蹤內回報! <br>
> [原作連結](https://github.com/Mooy1/SimpleUtils) | [非官方Discord](https://discord.gg/GF4CwjFXT9)

## 特徵

- 增強工作台 - 製作原版與黏液科技配方在單個選單

- 簡單電梯 - 蹲下來下去, 跳躍來上去

- 扳手 - 快速破壞物流節點, 電容, 和電力機器
